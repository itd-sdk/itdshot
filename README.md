# itdshot

Скриншотер ИТД.

![](example.png)
> Пример готового скриншота

## Установка
```bash
uv tool install itdshot
```
В будущем будет добавлена поддержка .exe (через pyinstaller).

## Использование
```bash
itdshot [ссылка на пост или id]
```
